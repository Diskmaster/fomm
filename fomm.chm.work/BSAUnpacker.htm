BSA unpacker
[Previous] [Main] [Next]

This utility is used for browsing BSA's and extracting files from them. Click 'open' and browse to a BSA of your choice to get started.

Select a single file and click 'preview' to preview it before extracting. (Only some file types can be previewed)

Click 'extract all' to extract the whole BSA to a folder of your choice.

Select one or more files, (multi-select is by ctrl/shift click, as with the rest of fomm,) and click 'extract' to extract just those files.

Choose a sorting mode and click 'sort' to sort the list of files.
